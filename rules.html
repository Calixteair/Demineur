Règles du démineur

Le démineur est un jeu de logique dans lequel le joueur doit découvrir toutes les cases vides d'une grille sans cliquer sur les cases contenant des mines.

Chaque case découverte indique le nombre de mines adjacentes. Le joueur peut utiliser ces informations pour déduire où se trouvent les mines et marquer les cases suspectes avec un drapeau.

Le jeu est gagné lorsque toutes les cases vides ont été découvertes. Si le joueur clique sur une mine, le jeu est perdu.
